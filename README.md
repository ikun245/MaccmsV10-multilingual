## 1. 多语言支持

### 1.1 根据IP地址自动显示语言
- 系统会根据访问者的IP自动检测其所在国家/地区，并自动切换对应的默认语言界面，例如中国用户默认中文，美国用户默认英文。
- 支持的语言可在后台管理中进行配置与扩展。

### 1.2 用户手动选择语言优先
- 页头或页脚提供语言选择控件，用户可自由切换所需语言。
- 若用户已选择语言，则优先以用户选择的语言显示内容，忽略IP识别结果。
- 多语言切换后，界面即时刷新，无需重新登录。
---

## 2. 邮件发送（SMTP）功能升级

- 系统内置SMTP邮件发送模块，支持账号注册、找回密码、消息通知等邮件功能。
- 后台支持配置SMTP服务器地址、端口、账号、密码、安全协议（SSL/TLS）等。
- 管理员可在后台测试邮件发送，实时查看邮箱投递情况。

---

## 3. 后台广告管理系统

### 3.1 广告位管理
- 可定义多个广告位（如首页顶部、播放页中部等），并指定展示顺序。
